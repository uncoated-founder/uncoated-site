# Uncoated

> See what's really inside.

**[Uncoated](https://uncoated.app)** is an independent product scanner that scores personal care and household products on two axes: ingredient safety and packaging sustainability.

Built for people who care about what's in the bottle and what the bottle's made of.

This repository hosts the public web pages for [uncoated.app](https://uncoated.app):

- **[Methodology](https://uncoated.app/methodology.html)** — how we score and where the data comes from
- **[Privacy Policy](https://uncoated.app/privacy.html)**
- **[Terms of Service](https://uncoated.app/terms.html)**

## Independence

Uncoated accepts no advertising, no brand sponsorship, and no paid product placement. Scoring is funded entirely by user subscriptions.
